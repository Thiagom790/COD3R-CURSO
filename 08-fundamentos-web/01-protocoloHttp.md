# Protocolo HTTP

## Requisições GET

Os dados vão direto nos parâmetros por url

## Requisições POST

Os dados costumam ir no body, e quando são senhas é bom criptografar

## Grupos de status

- 1xx - Informação
- 2xx - Sucesso
- 3xx - Redirecionamento
- 4xx - Erro no cliente
- 5xx - Erro no servidor
